■ 6. CSS 선택자를 스스로 알아내는 방법
모든 링크의 기본 색 : 검은색
방문한 적 있는 링크 : 회색
현재 페이지 : 빨간색

먼저 HTML 태그의 class를 saw로 지정.
선택자를 saw로 하면 saw라는 태그를 가리키는 선택자가 됨.
선택자를 .saw로 하면 class가 saw인 태그를 가리키는 선택자가 됨.
(class : 그루핑하다, 하나로 묶는다.)

현재 머물고 있는 페이지
class에 active 추가.(class="saw active")
선택자 .active 추가.

class : 여러 개의 값이 가능, 띄어쓰기로 구분.
하나의 태그에는 여러개의 속성 가능, 여러 개의 선택자를 통해서 하나의 태그를 공동으로 제어할 수 있음. 

같은 선택자일 경우 순서에 따라서 참조한다. 보다 나중에 있는 명령이 적용된다.

다른 선택자일 경우
하나를 id값으로 지정해보자.

#active
.saw

class="saw" id="active"

id선택자 > class선택자 > tag선택자
#val {} > .val {} > a {}

Q. 왜 이런 우선순위일까? 의문을 가져야 한다.
id 선택자의 값은 단 한번만 등장해야 한다.(학교에서 id는 학번, 국가에서는 주민등록번호 등 id의 핵심은 중복되서는 안된다 이다)
좀 더 구체적인 것의 우선순위 > 포괄적인 것의 우선순위.
디자인하고 코딩할 때 훨씬 효율적이다.

태그 = element
 
영문으로 뭐라고 하는지를 잘 알아야 할듯. 그래야 검색해서 찾아서 쓰기 편할 것이다.
